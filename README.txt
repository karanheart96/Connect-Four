Connect-Four
------------

Requirements:
-------------
Python 3
PyQt5 (if not present, pip install PyQt5)


How to run the code:
--------------------
python main.py


Components:
-----------
1. main.py - The starting point of the code and houses the main menu GUI.
2. board.py - Contains metadata of game board and functions related to it
3. game.py - Contains logic for choosing bot, logic for Running game GUI
4. bot.py - contains the logic for AI bots
